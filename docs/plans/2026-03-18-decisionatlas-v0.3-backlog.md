# DecisionAtlas v0.3 Backlog

## Priority Order

1. GitHub App auth and webhook-based incremental sync
2. private repository support with safer secret handling
3. login, reviewer/admin roles, and workspace scoping
4. multi-workspace management
5. richer benchmark corpus from a real public demo repo

## Explicitly Not in v0.2

- GitHub App installation flow
- organization-level permission matrix
- Slack, Jira, or Linear native connectors
- SaaS billing or multitenant tenancy model
- automated remediation suggestions

## Entry Criteria

Start v0.3 only after:

- v0.2 public demo is stable
- live provider path is exercised end-to-end
- import job tracking is reliable
- documentation and smoke checks stay green
