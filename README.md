# DecisionAtlas

DecisionAtlas is a self-hosted decision memory platform for engineering teams.

The MVP focuses on:

- importing GitHub and docs content
- extracting candidate decisions
- answering why-questions with citations
- detecting first-pass drift from accepted decisions
